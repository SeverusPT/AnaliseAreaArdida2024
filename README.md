
# Análise da área ardida em Portugal continental no ano de 2024

Na segunda quinzena de 2024 Portugal enfrentou incêndios de elevada magnitude com consequências 
ainda por apurar nas populações locais, no ambiente, nos serviços dos ecossistemas e na 
biodiversidade.

Tendo por base colmatar algumas lacunas de informação sobre este fenómeno, o Projeto SeverusPT 
preparou um relatório preliminar com estatísticas atualizadas sobre o impacto dos incêndios de 
2024. Acreditamos que esta informação poderá ser de interesse para uma melhor compreensão dos 
impactos dos incêndios e para aclarar algumas questões subjacentes a este fenómeno, recorrendo 
a factos fundamentados pelos dados mais atuais e de referência a nível nacional e europeu.

Link relatório: [https://severuspt.github.io/AnaliseAreaArdida2024](https://severuspt.github.io/AnaliseAreaArdida2024)

Todas as análises que constam do relatório são baseadas em fontes abertas de dados listadas 
no documento. Os processos de análise estatística e geoespacial dos dados são transparentes e 
encontram-se disponíveis neste repositório de código aberto.

Estas análises são preliminares e sujeitas a mudança assim que forem libertados mais dados 
sobre os eventos recentes, imagens de satélite ou validações do terreno que permitirão avaliar 
a severidade dos danos nas comunidades e ecossistemas afetados de forma mais apurada e precisa.
